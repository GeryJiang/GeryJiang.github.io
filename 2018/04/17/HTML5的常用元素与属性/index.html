HTML5常用的元素与属性

发表于 2018-04-17 | 分类于 HTML5

HTML5保留的常用元素

基本元素

  • <!-- -->: 定义注释
  • <html>: 文档根元素
  • <head>: 定义文档头部信息
  • <title>: 定义文档页面标题
  • <body>: 定义文档的主题部分
  • <h1>~<h6>: 定义标题
  • <p>: 定义段落
  • <br>: 插入一个换行
  • <hr>: 定义水平线
  • <div>: 定义文档中的节
  • <span>: 定义一般性文本

文本格式相关元素

  • <b>: 粗体
  • <i>: 斜体
  • <em>: 强调文本，实际效果与斜体差不多
  • <strong>: 粗体，与<b>基本相同
  • <small>: 小号字体文本
  • <sup>: 上标
  • <sub>: 下标
  • <bdo>: 定义文本显示方向，应指定dir属性，值为ltr或rtl（指标签内的文本在显示时从哪个方向开始显示）。

语义相关元素

  • <abbr>: 缩写，指定title属性代表全称
  • <address>: 地址，通常会被斜体显示
  • <blockquote>: 定义一段长的引用文本
  • <q>: 短引用文本
  • <cite>: 表示作品的标题（歌名书名等），斜体显示
  • <code>: 表示一段计算机代码
  • <dfn>: 定义一个专业术语，通常斜体或粗体显示
  • <del>: 删除线,中画线显示
  • <ins>: 插入的文本，下画线显示
  • <pre>: 预格式
  • <samp>: 定义示范文本内容
  • <kdb>: 键盘文本，表示文本是键盘输入的
  • <var>: 表示一个变量，斜体显示

使用a元素添加超链接和锚点

a元素的属性：

  • href：指定超链接所链接的另一个资源（url）
  • hreflang：指定超链接所链接的的文档所使用的语言
  • target：指定使用哪个框架来装载另一个资源，该属性有_self（当前窗口），_blank（新窗口），_top（顶层框架），_parent（父框架）。
  • download：用于让用户下载链接指定的资源，属性值为下载时的默认文件名
  • type：指定链接文档的MIME类型
  • media：指定目标链接所引用的媒体类型
    a元素主要可以包含文本，图像，各种文本格式化元素和表单元素等内容。

列表元素

  • <ul>: 无序列表
  • <ol>: 有序列表
    • start：指定列表的起始数字
    • type：指定使用哪种类型的编号，1代表数字，A或a代表使用大小写字母，I或i代表使用大小罗马数字（已不推荐，可用css修饰）
    • reversed：定义排序反转
  • <li>: 定义列表项
  • <dl>: 定义术语列表
  • <dt>: 定义标题列表项
  • <dd>: 定义普通列表项

使用img元素添加图片

img必须指定元素：

  • src：指定图片文件的位置
  • alt：指定一段文字，用于图片的提示信息
  • height：高
  • width：宽
    与图片相关有另两个元素：
  • <map>: 用于定义图片的映射，包含一个或多个<area>元素每个<area>元素定义一个区域，链接到不同的url。
  • <area>: 定义图片映射的内部区域，空元素，属性：
    • shape：指定内部区域是那种区域，默认rect，还可定义circle和ploy圆形与多边形区域。
    • coords：指定多个坐标值，用于确定区域的位置。
    • href：用于确定区域链接的资源
    • alt：同图片alt
    • target：同a标签

表格相关元素

  • <table>: 定义表格
    • cellpadding：单元格内容与单元格边框的距离
    • celllspacing：指定单元格之间的间距
  • <caption>: 表格标题，只可有0个或多个
  • <tr>: 定义表格行
  • <td>: 定义单元格
    • colspan: 指定单元格跨多少列
    • rosspan: 指定单元格跨多少行
  • <th>: 定义表格的表头单元格
  • <thead>: 定义表格头
  • <tbody>: 定义表格主体
  • <tfoot>: 定义表格脚
    使用<tbody>可以将表格分成几个独立的部分。
  • <col>: 为表格中的一个或多个列指定属性值
  • <colgroup>: 为表格中的一个或多个列指定属性值,用于组织多个<col>。

    增强的iframe元素

    iframe属性：
  • src： 指定url
  • name：指定该iframe名字
  • longdesc：指定url，包含该iframe的长描述
  • scrolling： 是否显示滚动条yes，no，auto。
  • height：高
  • width：宽
  • frameborder：是否显示边框
  • marginheight：上下页边距
  • marginwidth：左右页边距
  • srcdoc：srcdoc允许直接指定html片段，即直接在其属性值中编写html即可显示，并且优先级高于src。
  • seamless：指定iframe看上去是否是原文档的一部分，不再显示边框和滚动条。
  • sandbox：一个安全性属性，增加一系列额外限制
    • allow-forms：允许框架内的表单提交
    • allow-same-origin：与allow-forms一起使用，允许ajax交互否则拒绝
    • allow-scripts：允许加载框架内的js脚本
    • allow-top-navigation：允许框架内的超链接导航到父级窗口

HTML5保留的通用属性

  • id：为元素指定唯一的标识。
  • style：为元素指定css样式。
  • class：用于匹配css样式的class选择器。
  • dir：设置元素中内容的排列方向，有ltr和rtl两个属性值。
  • title：为元素指定额外信息，通常在鼠标移到该元素上时会显示title属性所指定的信息。
  • lang：设置该值告诉浏览器和搜索引擎网页与元素所使用的语言。
  • accesskey：设置激活元素的快捷键，通过alt+x（x为设置的accesskey的值）。
  • tabindex：设置值来通过tab键来不断切换窗口或页面中的html元素，来获得焦点。

HTML5新增的通用属性

  • contentEditable：该属性值如果为true，那么，浏览器将会允许开发者直接编辑该元素内的内容。具有“可继承”的特点，如果一个元素的父元素可编辑那么该元素默认也是可编辑的，除非指定contentEdiable为false。
  • designMode：designMode相当于一个全局的contentEditable。
  • hidden：支持true与false，若为true则该元素不会在浏览器上显示，且不保留占用空间。
  • spellcheck：为<input /><textarea/>等元素增加此元素，boolean值属性，可以对用户输入的文本内容进行输入检查，若检查不通过将会对拼错的单词进行提醒。
  • contentmenu：为元素设置上下文菜单，当用户右键单击是激发菜单。

HTML5新增的结构元素

  • <article>: 表示页面上完整独立的一篇文章。可用<header><footer>来进行定义文章的标题和脚注，也可用多个<section>分成多个段落。
  • <section>: 对页面内容进行分块。
  • <header>: 代表标题，类似hn标签，但使用更加广泛，可包含<p>和<span>等标签。
  • <footer>: 用于父级元素定义脚注信息，如版权等。
  • <nav>: 用于定义页面上的导航条
  • <aside>: 用来定义当前页面或文章的附属信息。
  • <main>: 一个html最多只能包含一个main元素，该元素用于包含除导航条，logo，版权信息等之外的主要内容。
  • <figure>: 用于表示一片独立的“图片区域”，内部包含一个或多个<img>标签。
  • <figcaption>: 定义<figure>的标题。

HTML5新增的语义元素

  • <mark>: 用于显示页面中需要重点“关注”的内容，通常被黄色显示出内容。
  • <time>: 用于显示被标注是日期或时间的内容。
  • <details>: 用于显示一段详细信息或某个主题的细节。
  • <summary>: 定义摘要信息，通常在<details>内部使用，摘要可见，点击摘要是显示details内容。
  • <ruby>: 用于为一个或多个短语定义ruby注释，短语后紧跟一段注释文字。
  • <rb>: 定义ruby注解所解释的词语
  • <rt>: ruby注解的说明部分（发音或其他注解）
  • <rp>: 用于为不支持ruby注解提供备用文本
  • <bdi>: 用于将一段文本从上下文中隔离出来，防止不同文字等导致的页面混乱
  • <wbr>: 用于指定在文本的指定位置添加换行
  • <menu>: 定义菜单
  • <menuitem>: 定义菜单项
    属性：
    type：定义菜单项的类型。支持command，checkbox，radio三个属性值
    label： 该属性指定菜单项的文本
    icon： 指定菜单项的图标
    disabled：指定菜单项是否不可用
    checked：指定菜单项是否已勾选
    radiogroup：用于指定该菜单项所属的菜单组
    default：指定该菜单项是否为默认菜单项

HTML5头部和元信息

  • <script>: 用于包含javascript脚本
  • <noscript>: 用于向禁止了javascript脚本或不支持javascript脚本的浏览器显示提示信息
  • <style>: 用于定义内部css样式
  • <link>: 用于链接图标，css样式以及各种外部资源
  • <title>: 用于定义文档标题
  • <base>: 用于指定页面中所有链接的基准路径
  • <meta>: 用于定义文档页面的元数据

<link>元素属性

  • href：链接资源url
  • hreflang： 链接资源的语言
  • media：设置链接资源仅适用与哪些设备
  • rel：设置文档与链接资源的关系
    rel属性值：
    alternate：链接资源是本文档的替代版本
    author：链接资源是本文档的作者
    copytight：；链接资源是关于本文档的版权信息
    help：链接资源是本文档的帮助
    icon：链接的图标是本文档的图标
    license：；链接资源是本文档的授权信息
    prefetch：用于预加载资源
    stylesheet： 外部样式单
  • sizes：指定图标的大小
  • type：指定链接资源的MIME类型

<base>元素属性

  • href：指定所有链接的基准路线
  • target：如a标签

<meta>元素属性

  • http-equip：指定元信息的名称（浏览器识别）
  • name：指定元信息的名称，可随意指定
  • content：指定元信息的值
  • charset：指定页面的字符集
  • expires：指定网页的过期时间
  • pragma：指定禁止浏览器从本地磁盘缓存中湖区该页面内容，浏览器一旦离开该网页就无法脱机访问
  • refresh：指定浏览器多长时间自动刷新指定页面
  • set-cookie：设置cookie
  • content-type：设置该页面的内容类型和所用的字符集

HTML5新增的拖放API

  <a><img>都是可拖动的，不过a需要设置href属性。而其他元素要实现拖动只需要把该新元素的draggable属性设置为true就可以了。但是这样拖动并没有携带数据，所以看不到拖动效果。因此需要拖动元素的ondragstart事件指定监听器，在该监听器中让拖动操作可以携带数据，这样就可以看到拖动效果了。

拖放操作相关事件

  • ondragstart：开始拖动操作时触发
  • ondrag：拖动过程中会不断的触发
  • ondragend：拖动结束时触发
  • ondragenter：被拖动的元素进入本元素范围内时触发
  • ondragover：拖动元素进入本元素范围内拖动时会不断触发
  • ondragleave：拖动元素离开本元素时触发
  • ondrop：其他元素被放到本元素时触发

DataTransfer对象

  • dataTransfer.dropEffect: 设置或返回拖放目标上允许发生的拖放行为。
  • dataTransfer.effectAllowed: 设置或返回被拖放元素允许发生的拖放行为。
  • dataTransfer.items: 该属性返回了dataTransferItems对象，该对象代表了拖动数据。
  • dataTransfer.setDragImage(element,x,y):设置拖放操作的自定义图标。
  • dataTransfer.addElement(element): 添加自定义图标。
  • dataTransfer.types: 该属性返回一个domstringlist对象，该对象包括了存入datatransfer中数据的所有类型。
  • dataTransfer个体DATa(format):获取datatransfer对象中的format格式数据。
  • dataTransfer.setData(format,data): 向datatransfer对象中设置format格式的数据。
  • dataTransfer.clearData([format]): 清除datatransfer对象中format格式的数据。
