样式单的基本使用

链接外部样式

引入外部样式通过在<head../>内插入<link…/>元素引入外部css样式单文档：



style=”property1:value1;property2:value2;property3:value3..”

伪元素选择器

  • :first-letter: 对指定对象内的第一个字符起作用，仅对块元素有效，对行内元素需要设置其width和height，或position为absolute，或display:block。
  • :first-line: 对指定对象的第一行内容自作用，与first-letter类似，只是显示作用效果时一行。
  • :before: 与内容相关属性相结合使用，用于在指定对象的内部的前端插入内容。
  • :after: 与内容相关属性相结合使用，用于在指定对象的内部的尾端插入内容。

内容相关属性

  • include-source: 插入值为url。
  • content：值为字符串，对元素之前或之后插入内容。除字符串外还可以使用url（“图片url”）格式插入图片。
  • quotes：用于为content属性定义open-quote和close-quote，两个值以空格分隔。content属性值应用open-quote和close-quote进行插入。
  • counter-increment：定义一个计数器，值为计数器名称。可自定义编号格式为content:counter（计数器名称，list-style-type），list-style-type的值就不列举了，可自行查找。
  • counter-reset：对指定的计数器复位。
  • 其他省略，有兴趣自行查找

CSS3新增的伪类选择器

结构性伪类选择器

  • selector:root: 匹配根元素
  • selector:first-child: 匹配选择器第一个子元素节点。
  • selector:last-child: 匹配选择器最后一个子元素节点。
  • selector:nth-child(n): 匹配选择器第n个子元素节点。
  • selector:nth-last-child(n): 匹配选择器倒数第n个子元素节点。
  • selector:only-child: 匹配选择器且作为父节点的唯一的子元素节点
  • selector:first-of-type: 匹配与选择器元素同类型同级的兄弟元素的第一个元素。
  • selector:last-of-type: 匹配与选择器元素同类型同级的兄弟元素的最后一个元素。
  • selector:nth-of-type(n): 匹配与选择器元素同类型同级的兄弟元素的第n个元素。
  • selector:nth-last-of-type(n): 匹配与选择器元素同类型同级的兄弟元素的倒数第n个元素。
  • selector:only-of-type: 匹配与选择器同级别与类型的兄弟元素中的唯一元素。
  • selector:empty: 匹配选择器内部没有任何子元素的元素。
  • selector:lang(lang): 匹配选择器其内部内容是特点语言的元素。
  • selector:nth-child(odd/event): 匹配选择器是其父节点的第奇数/偶数个节点的元素。
  • selector:nth-last-child(odd/event): 匹配选择器是其父节点的倒数第奇数/偶数个节点的元素。
  • selector: nth-child(xn_y): 匹配选择器必须是其父节点的第xn+y个子节点的元素。
  • selector:nth-last-child(xn+y): 匹配选择器必须是其父节点的倒数第xn+y个子节点的元素。
  • selector:nth-of-type(odd/event): 匹配选择器相同级别与类型的第奇数或偶数个元素。
  • selector:nth-last-of-type(odd/event): 匹配选择器相同级别与类型的倒数第奇数或偶数个元素。
  • selector:nth-of-type(xn+y): 匹配选择器同类型与级别的第xn+y个元素。
    • selector:nth-last-of-type(xn+y): 匹配选择器同类型与级别的倒数第xn+y个元素。

UI元素状态伪类选择器

  • selector:link: 匹配选择器且未被访问前的元素（通常只能是超链接）
  • selector:visited: 匹配选择器且已被访问的元素（通常只能是超链接）
  • selector:active: 匹配选择器处于被用户激活（鼠标点击或释放之间的事件）状态的元素
  • selector:hover: 匹配选择器鼠标悬停状态的元素
  • selector:focus: 匹配选择器且已得到焦点的元素
  • selector:enabled: 匹配选择器且当前处于可用状态的元素
  • selector:diasabled: 匹配选择器且当前处于不可用状态的元素
  • selector:checked: 匹配选择器且处于选中状态的元素
  • selector:default: 匹配选择器且页面打开时处于选中状态的元素
  • selector:indeterminate: 匹配选择器且处于选中状态不明确的元素
  • selector:read-only: 匹配选择器且处于只读状态的元素
  • selector:read-weite: 匹配选择器且处于读写状态的元素
  • selector:required: 匹配选择器且具有必填要求的元素
  • selector:optional:匹配选择器且无必填要求的元素
  • selector:valid: 匹配选择器且能通过校验的元素
  • selector:invalid: 匹配选择器且不能通过校验的元素
  • selector:in-range: 匹配选择器且处于指定范围内的元素
  • selector:out-of-range: 匹配选择器且当前超出指定范围的元素
  • selector::selection: 匹配选择器中当前被选中的内容

浏览器专属属性

有些CSS属性时最新版的，还未发布，为了让浏览器识别这些专属属性，Css规范允许在属性前添加专属前缀，常见的前缀有：

  • -ms-：IE专属
  • -moz-：基于Gecko引擎的浏览器，如firefox
  • webkit-：基于WebKit引擎的浏览器，如Chrome，Safari，Opera

:target伪类选择器与:not伪类选择器

  • selector:target: 符合选择器，且必须是命名锚点目标的元素
  • selector1:not(selector2): 符合1但不符合2的选择器
