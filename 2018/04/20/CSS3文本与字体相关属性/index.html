CSS3文本与字体相关属性

发表于 2018-04-20 | 分类于 CSS3

字体相关属性

  • font: 字体符合属性。
  • color: 字体颜色
  • font-family: 设置文字字体
  • font-size: 设置字体大小
  • font-size-adjust: 控制不同字体的字体尺寸，值为字体的aspect值，aspect值为该字体的大小除以该字体小写字母x的高度。
  • font-stretch: 改变字体的横向拉伸，normal，narrower，wider三种值。
  • font-style: 设置文字风格，常用值有normal，italic，oblique等。
  • font-weight: 加粗，lighter，normal，bold，bolder等，也可以具体数值。
  • text-decoration: 文字修饰线，none，blink(闪烁)，underline，line-through，overline等。
  • text-variant: 设置文字的大写字母格式。normal，small-caps。
  • text-transform: 设置文字的大小写，none，capitalize，uppercase，lowercase。
  • line-height: 设置字体行高，设置负值的行高可以实现阴影效果，设置为与字体容器一样的高度则可以使文字居中。
  • letter-spacing: 设置字符之间的间隔，指定间隔添加到每个字符之后，最后一个不受影响。
  • word-spacing: 设置但系之间的间隔。

text-shadow:

  • color：指定阴影颜色
  • xoffset：横向上的偏移
  • yoffset：纵向上的偏移
  • radius：指定阴影的模糊半径，值越大越模糊。
  • 可添加多个阴影，每组值用，隔开。

文本相关属性

  • text-indent: 设置段落文本的缩进。
  • text-overflow: 用于控制溢出文本的控制方法。clip只裁剪，ellipsis显示溢出标记。
  • text-align: 设置文本的水平对齐方式。
  • direction: 设置文本的流入方向。
  • white-space: 设置对文本中空白的处理。
  • word-break: 设置文本的换行方式。
  • word-wrap: 设置文本的换行方式。
