HTML5的绘图支持

发表于 2018-04-18 | 分类于 html5

canvas对象

<canvas>元素是专门用来绘制图形的，但是它只是一个绘图容器，即相当于一张画布，绘图步骤如下：

  • 调用canvas对象的dom对象
  • 调用canvas的dom对象的getContext（）方法，该方法返回一个canvasRenderingContext2D对象，该对象即可绘图
  • 调用canvasrenderingcontext2d对象的方法进行绘图

绘制几何图形

  • fillRect(double x,double y,double width, double height):填充一个矩形区域
  • strokeRect(double x,double y,double width, double height):绘制一个矩形边框

    绘制字符串

  • fillText（String text，double x，double y，[double maxwidth]）:填充字符串
  • strokeText（String text，double x，double y，[double maxwidth]）:绘制字符串边框
    属性：
  • textAlign：水平对齐方式
  • textBaseAlign：垂直对齐方式

设置阴影

阴影属性：

  • shadowBlur：设置阴影模糊度
  • shadowcolor：阴影颜色
  • shadowoffsetx：阴影x方向的偏移
  • shadowoffsety：阴影y方向的偏移

使用路径

canvas路径方法步骤：

  • 2d对象调用beginPath()方法开始定义路径
  • 2d对象的各种方法添加子路径
  • 2d对象的closepath()方法关闭路径
  • 2d对象的fill()或stroke()方法填充路径会绘制边框
    提供的基本方法：
  • arc(double x,double y,double radius,double startangle,double endangle,boolean countercloclwise):当前路径上添加一段弧
  • arcTo(double x1,double y1,double x2,double y2,double radius):当前路径上添加一段弧,与上一方法相比只是定义弧的方式不同而已
  • lineTo(double x,double y)：从当前结束点连接到x，y对应的点
  • moveTo(double x,double y)：从当前结束点移动到x，y对应的点

绘制曲线

  • bezierCurveTo(double cpx1,double cpy1,double cpx2,double cpy2,double x,double y): 贝济埃曲线，四个点两个控制点一个开始点一个结束点
  • quadraticCurveTo(double cpx,double cpy, double x,double y):三个点，开始，控制与结束

绘制位图

  • drawimage(image image,double x,double y): 把image会知道x，y处，不对图片做任何处理
  • drawimage(image image,double x,double y，double width,double height): 给定宽高绘制到x，y处
  • drawimage(image image,integer sx,integer sy,integer sw,integer sh,double dx,double dy,double dw,double dh):从image上挖出一块放到canvas上，sx和sy为哪个位置开始挖，sw和wh定义挖的宽高，dx和dy为绘制到canvas哪个位置，dw和dh设置宽高

坐标变换

  • translate(double dx,double dy): 平移dx和dy
  • scale(double sx,double sy): 缩放，大小为sx，sy对比值，值都为1，则无变化
  • rotate(double angle): 旋转，旋转angle角度
  • transform(double m11,double m12,double m21,double m22,double dx,double dy):对当前坐标系统执行矩阵变化
  • save(): 保存当前的绘图状态
  • restore(): 回复之前保存的绘图状态
