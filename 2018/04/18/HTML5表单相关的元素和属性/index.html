HTML5表单相关的元素和属性

发表于 2018-04-18 | 分类于 html5

表单及表单控件

form元素及属性

  • 用于生成输入表单，不会生成可视化部分，其他表单元素需要放在该标签内。
  • action：指定当单击表单内的“确认”按钮时，该表单被提交到哪个地址。
  • method：指定提交表单时发送何种类型的请求，可为get和post，通常建议post。
  • enctype：指定对表单内容进行编码所使用的字符集。
  • name：指定表单的唯一名称，建议与id相同。
  • target：同超链接target一致。

get方式请求：

get方式的请求会将请求参数的名称和值转换成字符串，并附加在原url之后，因此地址栏可以看到请求参数的名和值，且get请求的传送数据量较小，一般不能大于2kb。

post方式请求

post请求的传送数据量较大，通常认为大小不受限，但往往取决于服务器的限制。post发送的请求参数放在html header中传输，无法在地址栏中看到，安全性相对较高。

关于表单控件转换成请求参数的规则

  • 每个name对应一个请求参数，无name则不会生成请求参数。
  • 控件同名，则生成一个请求参数，但该参数有多个值。
  • name指定参数名，value指定参数值。
  • 若控件设置了disabled属性，则不在生成请求参数。

input元素及属性

type属性值：

  • text：单行文本框
  • password：密码输入框
  • hidden：隐藏域
  • radio：单选框
  • checkbox：复选框
  • image：图像域，当为此值时，input可设置宽高。
  • file：文件上传域
  • submit：提交按钮
  • reset：重设按钮
  • button：无动作按钮
    h5新增
  • color：颜色选择器
  • date：日期选择器
  • time：时间选择器
  • datatime-local：本地日期，时间选择器
  • week：选择第几周的文本框
  • month：月份选择器
  • email：e-mail输入框，自动检测邮箱格式
  • tel：电话号码文本框
  • url：url输入框，自动检测格式
  • number：输入一个只能输入数值的文本框，有min，max，step，valueasnumber属性
  • range：生成一个拖动条，min，max，step属性
  • search：生成一个专门用于输入搜索关键词的文本框
  • checked属性：设置选框初始状态是否为选中，type为radio或checkbox才可用
  • disabled属性：加载时禁用此元素
  • maxlength属性：文本框允许输入的最大字符数
  • readonly：指定文本框不允许用户修改
  • size：指定该元素的宽度
  • src：指定图像域显示图像的url

    label标签

  • 隐式使用for属性：用for属性值关联控件id属性值。
  • 显示关联：将普通文本域表单控件一起放在label内。
    都可通过文本获取控件焦点

使用button定义按钮

  • disabled：是否禁用
  • name：按钮唯一名称
  • type：定义属于哪种按钮（submit，reset，button）
  • value：按钮的初始值

select和option元素

select用于创建列表框或下拉菜单，与option一起使用，一个option代表一个列表项或菜单项

  • disabled：禁用列表框和下拉菜单
  • multiple：设置列表框或下拉菜单是否允许多选
  • size：指定列表框内可同时显示多少个列表项
    **是生成列表框还是下拉菜单，是由是否指定size或multiple属性决定的，若具有这两个属性之一，则为列表框，否则为下拉菜单。
    <option>用于定义列表框选项或菜单选项
  • disabled：禁用该选项
  • selected：默认被选中
  • value：选项对应的请求参数的值
    <optgroup>用于定义列表项或菜单项组
  • label：指定选项组的标签
  • disabled：禁用该选项组的所有选项

textarea元素

此元素用于生成多行的文本域。

  • cols：指定文本域的宽度
  • rows：指定文本域的高度
  • disabled：是否禁用
  • readonly：只读
  • maxlength：设置最多输入的字符数
  • wrap：指定多行文本域是否添加换行，支持soft和hard值。若为hard则必须指定cols值。

fieldset和legend元素

<fieldset>对表单内元素进行分组

  • name：元素名称
  • form：值为一个有效的form表单id，指定fieldset属于该表单
  • disabled：禁用该组表单元素
  • <legend>用于设置<fieldset>标题

HTML5新增的表单属性

  • form属性: 用于定义该表单控件所属的表单，值为表单id。该属性使表单控件可以定义在form标签外，但提交时，依然有效。
  • formaction：设置按钮提交的url。
  • formXXX：动态的设置按钮的一些属性
  • autofocus：自动获取焦点。一个页面只能有一个控件有此属性
  • placeholder：为文本框设置提示信息，当开始输入时自动消失
  • list：与datalist元素结合使用，其相当于一个看不见的select，只有在双击有list属性且值时datalist的id值时才会显示。
  • autocomplete：设置表单是否支持自动完成功能，即根据上次提交的数据生成列表框供用户选择
  • label标签新增的control属性：用于js访问label标签关联的表单元素

HTML5新增的表单元素

output元素

用于显示输出，且只用于显示输出，并不生成请求参数，for属性用于指定将会显示哪个或哪些元素的值。

meter元素

用于表示一个已知的最大值和最小值的计数仪表

  • value：指定计数仪表的当前值
  • min：指定技术仪表的最小值
  • max：最大值
  • low：指定范围的最小值，大于都等于min
  • high：指定范围的最大值，小于等于max
  • optimum：指定计数仪表有效范围的最佳值

progress元素

表示一个进度条

  • max：指定进度条完成时的值
  • value：指定进度条当前完成的进度值

keygen元素

用于生成公钥和密钥对，提交表单时私钥存储本地，公钥发送到服务器

  • name：名称
  • keytype：指定生成公钥密钥对的算法。
  • disabled：是否禁用

HTML5新增的客户端校验

使用校验属性执行校验

  • required：指定表单控件必须填写
  • pattern：指定控件的值必须符合正则表达式
  • min：最小值
  • max：最大值
  • step：步长

调用checkvalidity方法进行校验

  • 如果表单对象调用该方法返回true，则表单内的所有表单元素都有效，且可以通过校验。

自定义错误提示

设置setCustomValidity（）方法显示错误提示信息。

关闭校验

  • 为form元素添加novalidate属性
  • 提交按钮添加formnovalidate属性
